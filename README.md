# XYZ Help Menu

Modern and clean FiveM help menu with a simple `/help` command.

## Description
XYZ Help Menu is a lightweight and easy-to-use FiveM help panel made for servers that want to show important information in one clean place.  
It includes keyboard binds, useful commands, server rules and general server information inside a modern tab-based UI.

## Features
- Clean and modern NUI design
- Open and close with `/help`
- Keyboard binds tab
- Useful commands tab
- Server rules tab
- Server information tab
- Easy to configure
- Custom title, subtitle and accent color
- Lightweight client-side logic
- Simple and organized layout
- Easy to edit for any server

## Configuration
You can easily change:
- Command name
- UI title
- UI subtitle
- Accent color
- Keyboard binds
- Commands
- Rules
- Server information
- Extra links such as Discord, website, store or social media
